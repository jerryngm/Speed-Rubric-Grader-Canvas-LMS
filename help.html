Welcome to Speed Rubric Grader!

Grade multiple Canvas submissions quickly using rubrics with bulk actions.

Quick Start Guide

Get up and running in just a few steps.

  1. Navigate to SpeedGrader

    Open any assignment in Canvas SpeedGrader. The extension button will appear automatically.

  2. Open the Extension

    Click the "Speed Rubric Grader" button to open the grading modal.

  3. Select Sections (Optional)

    Use the section filter to narrow down which students to load. This is helpful for large courses.

  4. Load Data

    Click "Load Data" to fetch all student submissions and rubric criteria.

  5. Grade in Bulk

    Select rubric ratings for multiple students at once. Use the search and filter tools to find specific students.

  6. Save Changes

    Click "Save Changes" to submit all grades back to Canvas.

Pro Tips
  • Section preferences are saved: Your selected sections are remembered per course
  • Use keyboard shortcuts: Tab through cells for faster navigation
  • Toggle comments: Show/hide the comments column to focus on grades
  • Search students: Use the search box to quickly find specific students
Important Notes
  • Changes are not saved until you click "Save Changes"
  • The extension requires SpeedGrader to be open on a rubric-enabled assignment
  • All data processing happens locally in your browser

Features

Bulk Grading

Grade multiple students at once by selecting rubric ratings in a spreadsheet-like interface.

Section Filtering

Filter students by section to focus on specific groups. Preferences are saved per course.

Search & Filter

Quickly find students using the search box. Filter by name or other criteria.

Comments Support

Add or edit rubric comments for each criterion. Toggle visibility as needed.

Settings Backup

Export and import your preferences. Transfer settings between browsers easily.

Keyboard Navigation

Navigate efficiently using Tab and arrow keys for faster grading workflows.
